Bem-vindo!

Este painel centraliza o acesso a todos os relatórios, APIs e informações do sistema de autocura cognitiva. Utilize o menu acima para navegar entre as funcionalidades.

  • Relatório Completo (JSON): Dados detalhados para integrações e análises avançadas.

  • Ações: Ações corretivas recomendadas e seu status.
  • Status dos Serviços: Saúde dos módulos do sistema.
  • Documentação: Guia de uso, arquitetura e implantação.
